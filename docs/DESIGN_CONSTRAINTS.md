## Cosmic Palette

The Cosmic Palette is the foundational color system for Advanced Knock. All colors must come from this palette.

### Colors

- `#0a0a12` - **void** - Deep space, background
- `#FFF8DC` - **light-source** - Pure light, highlights
- `#FF69B4` - **freq-love** (528 Hz) - Love, connection, emotion
- `#9B59B6` - **freq-truth** (777 Hz) - Truth, clarity, wisdom
- `#6366f1` - **freq-power** (999 Hz) - Power, action, execution
- `#10B981` - **freq-growth** - Growth, progress, success

### Usage Rules

- Use void for backgrounds and deep spaces
- Use light-source for highlights and emphasis
- Map emotions to frequencies:
  - Love/Connection → freq-love (#FF69B4)
  - Truth/Clarity → freq-truth (#9B59B6)
  - Power/Action → freq-power (#6366f1)
  - Growth/Success → freq-growth (#10B981)

## Typography Duality

Two voices for two purposes: Soul Voice for emotion, Clear Voice for function.

### Soul Voice

**Contexts:** heading, emotion, celebration

- Used for emotional content
- Headlines, hero text, celebrations
- Conveys feeling and connection

### Clear Voice

**Contexts:** body, ui, data

- Used for functional content
- Body text, UI labels, data display
- Conveys clarity and information

### Usage Rules

- Never use Soul Voice for functional content
- Never use Clear Voice for emotional content
- Match voice to context automatically

## Fibonacci Spacing

All spacing must follow the Fibonacci sequence for natural harmony.

### Spacing Scale

- `8px` - Tight spacing
- `16px` - Standard spacing
- `24px` - Comfortable spacing
- `32px` - Generous spacing
- `48px` - Section spacing
- `64px` - Large section spacing
- `96px` - Hero spacing

### Usage Rules

- Always use Fibonacci spacing values
- Never use arbitrary spacing values
- Scale spacing proportionally

## Living Canvas Protocol

The Living Canvas Protocol defines the structure of interactive experiences.

### Structure

1. **Emotion** → Pattern mapping
2. **Maximum 3 steps** to completion
3. **ONE action** per screen
4. **60fps** animation performance

### Usage Rules

- Map emotions to animation patterns
- Limit interactions to 3 steps maximum
- One primary action per screen
- Maintain 60fps performance

## Animation Pattern → Emotion Mapping

Animations must map to emotions using the Cosmic Palette frequencies.

### Mappings

- **Love** → freq-love (#FF69B4) → Gentle pulse, soft fade
- **Truth** → freq-truth (#9B59B6) → Clear reveal, sharp focus
- **Power** → freq-power (#6366f1) → Strong motion, confident action
- **Growth** → freq-growth (#10B981) → Expanding motion, upward flow

### Usage Rules

- Match animation to emotion
- Use color from Cosmic Palette
- Maintain 60fps performance
- Keep animations purposeful
