## Overview

The Advanced Knock Index repository serves as the coordination hub for the Advanced Knock and Advanced Ring multi-repo development ecosystem. This repository provides:

- **Centralized Coordination** - Single source of truth for cross-repo dependencies and relationships
- **Index System** - Alternative to Turbo Repo for multi-repo coordination
- **Unified Templates** - Issue and PR templates for all repositories
- **Cross-Repo CI/CD** - Workflows that coordinate builds across repositories
- **Documentation Hub** - Master documentation for architecture, workflows, and collaboration

## Quick Start

### Prerequisites

- Node.js 20+
- npm or pnpm
- Git
- Access to Advanced-Knock organization repositories

### Setup

```bash
# Clone the index repository
git clone https://github.com/Advanced-Knock/advanced-knock-index.git
cd advanced-knock-index

# Install dependencies
npm install

# Build index
npm run build
```

### Build All Repositories

```bash
# Build all repositories in correct order
./scripts/build-all.sh
```

### Sync Index

```bash
# Sync index files from all repositories
node scripts/sync-index.js
```
